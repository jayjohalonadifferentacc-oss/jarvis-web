# JARVIS

Local-first personal AI assistant. Memory is a folder of markdown; the brain is a live visualisation of it working. Runs locally and free on Ollama.

## Download

- **Windows:** see the site below (installer in `release/`)
- **Android:** APK in `release/` — a thin client that drives your PC's JARVIS over the LAN

Visit the site: https://jayjohalonadifferentacc-oss.github.io/jarvis-web/
